Was ist das?:

  • Protokoll zur Datenübertragung zwischen Client und Server
  • Ein Transportprotokoll
  • Wenn Paket erhalten: Nächstes Paket wird versendet
  • Wenn nicht: Paketverlust

Verbindungsaufbau:

Datenübertragung:

  • Daten werden in Pakete verpackt
  • Sender sendet Pakete an den Empfänger und wartet auf Bestätigung
  • Wenn Bestätigung erhalten: Nächstes Paket wird versendet
  • Wenn nicht: Paketverlust
    • Paket ist nicht angekommen
    • Jedes Paket hat eine Sequenznummer
      • dient für richtige Reihenfolge und Vermeidung von Duplikaten
    • Festellung, welches Paket verloren gegangen ist durch Sequenznummer
    • Erneuter Versuch das Paket zu versenden
